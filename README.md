# Software Design2023 年 7 月号「gRPC で始める Web API 開発」サンプルコード

Software Design 2023 年 7 月号の特集「gRPC で始める Web API 開発」、第 4 章「gRPC による Web API の実装」のサンプルコードです。

## 動作確認環境

- Node.js v18.16.0
- npm v9.6.4
- Buf CLI 1.19.0

## セットアップ

```
npm install
```
